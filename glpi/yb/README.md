Скрипт должен ходить с заданной переодичностью в базу данных, брать актуальный список сайтов, сравнивать его с предыдущем списокм и если были изменения, то создавать файл(ы) в соответствующих папках (папка создается по имени ПК).
Есть часть сайтов, которые созданы админом. Они создаются автоматически, а все остальные сайты дописываются после них.
Если было создание файлов, то запускается скрипт, который пробегает по этим папкам и обновляет файл из папки на конкретном ПК (ПК выбирается по имени папки)
